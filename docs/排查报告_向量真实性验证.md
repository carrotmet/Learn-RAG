| 指标 | 实际值 | FakeEmbeddings 特征 | 真实嵌入特征 |
|------|--------|---------------------|-------------|
| 维度 | 2048 | 2048 (可配置) | 2048 (nvidia/llama-nemotron-embed-vl-1b) |

**结论**: 所有向量值都在 0-1 之间，均值 ~0.5，完全符合 FakeEmbeddings 特征。

### 3. 代码根因分析

**问题文件**: `backend/src/hybrid/channels/vector.py`

**原代码（第31-35行）**:
```python
def _load_embedding(self):
    try:
        from agent.vector_store import OpenRouterEmbeddings
        return OpenRouterEmbeddings()  # ❌ 没有传参数！
    except Exception:
        pass  # 静默失败，fallback 到 FakeEmbeddings
```

**OpenRouterEmbeddings.__init__ 签名**:
```python
def __init__(self, model: str, api_key: str, ...):
```

**问题**:
1. `OpenRouterEmbeddings()` 不传参数 → `TypeError` → 被 `except Exception` 捕获
2. 没有加载 `.env` 文件 → `os.getenv("OPENROUTER_API_KEY")` 为 None
3. 静默 fallback 到 FakeEmbeddings，用户无感知
