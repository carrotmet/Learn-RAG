    description: "概念定义检索，回答'是什么'类问题"

  retrieve.relation:
    description: "关系图谱检索，查找实体间关联"
    tags: [retrieve, relation, graph]

    description: "全局技能调用，跨项目通用能力"

  skill.project:
    description: "项目内技能，依赖当前工程上下文"
    tags: [skill, project, context-aware]

  tool.report:
    description: "报表生成工具"
    tags: [tool, report, output]

# ==========================================
# 2. 倒排索引：供大模型 function call 检索
# ==========================================
index:
  by_tag:
    retrieve: [retrieve.keyword, retrieve.concept, retrieve.relation]
    skill: [skill.global, skill.project]
    tool: [tool.report]
    search: [retrieve.keyword, retrieve.concept]

  by_keyword:
    "查": [retrieve.keyword, retrieve.concept]
    "找": [retrieve.keyword]
    "是什么": [retrieve.concept]
    "生成": [tool.report]
    "报表": [tool.report]

# ==========================================
# 3. 大模型返回值规范（极简）
# ==========================================
response_schema:
  type: object
  properties:
    intents:
      type: array
      items:
        type: object
        properties:
          path:
            type: string
            description: "从注册表中选择的点分路径"
            enum: [retrieve.keyword, retrieve.concept, retrieve.relation, skill.global, skill.project, tool.report]
          confidence:
            type: string
            enum: [high, medium, low]
            description: "对该意图匹配的置信度"
          reasoning:
            type: string
            description: "简要说明选择理由，辅助调试"
        required: [path, confidence]
    primary_intent:
      type: string
      description: "主意图路径，当多意图冲突时以此为准"
  required: [intents]
```

如果意图不明确，请先调用 search_intents 工具缩小范围。
返回格式：intents 数组，每个元素包含 path, confidence, reasoning。
注意：你只需要识别意图，不需要提取任何实体或参数。
```

**Step 2（可选）：function call 检索**

```python
def search_intents(tag: str = None, keyword: str = None, max_results: int = 10) -> list:
    """基于倒排索引快速过滤候选意图"""
    results = set()
    if tag:
        results.update(index["by_tag"].get(tag, []))
    if keyword:
        results.update(index["by_keyword"].get(keyword, []))
    return [{"path": p, "description": intents[p]["description"]} for p in list(results)[:max_results]]
```
