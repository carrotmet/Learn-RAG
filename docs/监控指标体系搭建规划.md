## 1. 体系概述

### 1.1 核心闭环

教学项目的核心目标是跑通 **数据格式统一 → 离线导入 → 数据解析 → 数据集搭建 → RAGAS 测试 → 反馈优化** 的闭环。

```
┌─────────────────────────────────────────────────────────┐
│  核心闭环（Core Loop）                                    │
│                                                          │
│  数据格式统一 ──→ 离线导入 ──→ 数据解析 ──→ 数据集搭建   │
│       ↑                                          │       │
│       └────────── 反馈优化 ←── RAGAS 测试 ←─────┘       │
│                                                          │
│  在线采集（P1）：API 直写 SQLite，补充数据源               │
└─────────────────────────────────────────────────────────┘
```

### 1.2 监控分层

```
┌─────────────────────────────────────────────────────────┐
│  Layer 3: 测试反馈（Testing Feedback）                  │
│  - RAGAS 评估、可视化报告、优化建议                      │
├─────────────────────────────────────────────────────────┤
│  Layer 2: 测试集搭建（Test Set Construction）           │
│  - 数据格式统一、数据解析、数据集构建                    │
├─────────────────────────────────────────────────────────┤
│  Layer 1: 数据采集（Data Collection）                   │
│  - 离线上传（P0）、在线采集（P1）、批量导出（P2）        │
├─────────────────────────────────────────────────────────┤
│  Layer 0: 基础设施（Infrastructure）                    │
│  - SQLite 数据库、Python 环境、文件系统                  │
└─────────────────────────────────────────────────────────┘
```

### 1.3 核心原则

| 原则 | 说明 |
|------|------|
| **闭环优先** | 先跑通 格式统一→导入→解析→数据集→RAGAS→反馈 的完整闭环 |
| **轻量实现** | 教学项目不用 Kafka/Grafana/Prometheus，SQLite + Python 足够 |
| **渐进建设** | 先离线导入（P0），再在线采集（P1），最后自动化（P2） |
| **数据驱动** | 所有指标必须有对应数据集支撑，拒绝"感觉良好" |
| **成本可控** | 免费方案优先：RAGAS + SQLite + OpenRouter 免费额度 |

## 2. Phase 1：数据采集（轻量教学版）

> 教学项目定位：无需 Kafka 等重型中间件。API 调用时直接写入 SQLite，简单可靠。

### 2.1 采集方式分类

| 采集方式 | 说明 | 适用场景 | 优先级 |
|---------|------|---------|--------|
| **离线上传** | 手动上传历史数据、测试数据、标注数据 | 冷启动数据补充、批量历史数据导入 | **P0** |

### 2.3 在线采集（API 直写 SQLite）

#### 采集方式

教学项目采用最简方案：**API 调用时直接写入 SQLite**，无需消息队列。

| 方案 | 实现 | 优点 | 缺点 | 选用 |
|------|------|------|------|------|
| **同步写入** | API 处理流程中直接 INSERT | 简单、数据不丢 | 增加 API 延迟 ~5ms | 默认 |
| **异步写入** | 后台线程池写入 | API 零延迟 | 极端情况可能丢 1 条 | 可选 |

#### 采集内容

| 数据类型 | 内容 | 采集时机 | 存储表 |
|---------|------|---------|--------|
| **用户对话** | question, answer, contexts | 每次 API 调用后 | `conversations` |
| **检索结果** | retrieved_chunks, scores | 检索完成后 | `retrieval_logs` |
| **LLM 调用** | prompt, response, tokens | LLM 返回后 | `llm_calls` |
| **用户反馈** | thumbs_up/down, comment | 用户交互时 | `user_feedback` |

#### 轻量采集架构

```
用户请求
    │
    ▼
┌─────────────────┐
│  FastAPI 接口    │
│  /api/chat       │
└────────┬────────┘
         │
    ┌────┴────┐
    ▼         ▼
┌────────┐  ┌─────────────┐
│ 业务处理 │  │ 采集写入     │
│ 检索+生成│  │ SQLite INSERT│
│        │  │ (同步/异步)  │
└────────┘  └─────────────┘
                 │
                 ▼
           ┌──────────┐
           │  SQLite  │
           │ rag_data.db
           │          │
           │ ├─ conversations
           │ ├─ retrieval_logs
           │ ├─ llm_calls
           │ └─ user_feedback
           └──────────┘
```

```python
# backend/src/data_collection/sqlite_store.py

import sqlite3
import json
import uuid
from datetime import datetime
from typing import Optional, List
from contextlib import contextmanager

class SQLiteCollector:
    """轻量 SQLite 采集器 — 教学项目首选"""
    
    def __init__(self, db_path: str = "data/rag_data.db"):
        self.db_path = db_path
        self._init_tables()
    
    @contextmanager
    def _get_conn(self):
        conn = sqlite3.connect(self.db_path)
        conn.row_factory = sqlite3.Row
        try:
            yield conn
            conn.commit()
        finally:
            conn.close()
    
    def _init_tables(self):
        """初始化采集表"""
        with self._get_conn() as conn:
            conn.executescript("""
                CREATE TABLE IF NOT EXISTS conversations (
                    id TEXT PRIMARY KEY,
                    question TEXT NOT NULL,
                    answer TEXT,
                    contexts TEXT,  -- JSON 数组
                    ground_truth TEXT,
                    model_version TEXT,
                    timestamp TEXT DEFAULT CURRENT_TIMESTAMP,
                    source TEXT DEFAULT 'online_api',
                    metadata TEXT  -- JSON
                );
                
                CREATE TABLE IF NOT EXISTS retrieval_logs (
                    id TEXT PRIMARY KEY,
                    conversation_id TEXT,
                    query TEXT,
                    retrieved_chunks TEXT,  -- JSON
                    scores TEXT,  -- JSON
                    latency_ms INTEGER,
                    timestamp TEXT DEFAULT CURRENT_TIMESTAMP
                );
                
                CREATE TABLE IF NOT EXISTS user_feedback (
                    id TEXT PRIMARY KEY,
                    conversation_id TEXT,
                    feedback_type TEXT,  -- thumbs_up|thumbs_down|comment
                    content TEXT,
                    timestamp TEXT DEFAULT CURRENT_TIMESTAMP
                );
                
                CREATE INDEX IF NOT EXISTS idx_conv_time 
                    ON conversations(timestamp);
                CREATE INDEX IF NOT EXISTS idx_conv_source 
                    ON conversations(source);
            """)
    
    def save_conversation(self, 
                        question: str,
                        answer: str,
                        contexts: List[str],
                        ground_truth: Optional[str] = None,
                        model_version: str = "v1.0.0",
                        metadata: Optional[dict] = None) -> str:
        """保存对话记录 — 同步调用，约 5ms"""
        conv_id = str(uuid.uuid4())
        
        with self._get_conn() as conn:
            conn.execute("""
                INSERT INTO conversations 
                (id, question, answer, contexts, ground_truth, 
                 model_version, source, metadata)
                VALUES (?, ?, ?, ?, ?, ?, ?, ?)
            """, (
                conv_id,
                question,
                answer,
                json.dumps(contexts, ensure_ascii=False),
                ground_truth,
                model_version,
                'online_api',
                json.dumps(metadata or {}, ensure_ascii=False)
            ))
        
        return conv_id
    
    def save_feedback(self, 
                      conversation_id: str,
                      feedback_type: str,
                      content: Optional[str] = None) -> str:
        """保存用户反馈"""
        fb_id = str(uuid.uuid4())
        
        with self._get_conn() as conn:
            conn.execute("""
                INSERT INTO user_feedback 
                (id, conversation_id, feedback_type, content)
                VALUES (?, ?, ?, ?)
            """, (fb_id, conversation_id, feedback_type, content))
        
        return fb_id
    
    def get_recent_conversations(self, 
                                  limit: int = 100,
                                  source: Optional[str] = None) -> List[dict]:
        """获取最近对话 — 用于测试集构建"""
        with self._get_conn() as conn:
            if source:
                rows = conn.execute(
                    "SELECT * FROM conversations WHERE source = ? 
                     ORDER BY timestamp DESC LIMIT ?",
                    (source, limit)
                ).fetchall()
            else:
                rows = conn.execute(
                    "SELECT * FROM conversations 
                     ORDER BY timestamp DESC LIMIT ?",
                    (limit,)
                ).fetchall()
            
            return [dict(row) for row in rows]
    
    def export_to_testset(self, 
                          output_path: str,
                          min_feedback: Optional[str] = None,
                          limit: int = 1000):
        """导出到测试集文件（JSONL）"""
        import json
        
        with self._get_conn() as conn:
            query = """
                SELECT c.* FROM conversations c
                WHERE c.ground_truth IS NOT NULL
                ORDER BY c.timestamp DESC LIMIT ?
            """
            rows = conn.execute(query, (limit,)).fetchall()
        
        with open(output_path, 'w', encoding='utf-8') as f:
            for row in rows:
                record = {
                    "id": row['id'],
                    "question": row['question'],
                    "answer": row['answer'],
                    "contexts": json.loads(row['contexts'] or '[]'),
                    "ground_truth": row['ground_truth'],
                    "metadata": {
                        "timestamp": row['timestamp'],
                        "model_version": row['model_version'],
                        "source": row['source']
                    }
                }
                f.write(json.dumps(record, ensure_ascii=False) + '\n')
        
        return len(rows)
```

#### 导入代码示例

```python
# backend/src/testset/importer.py

import sqlite3
import json
import uuid
from typing import List, Dict, Optional
from datetime import datetime

class DataImporter:
    """离线数据导入器 — 统一格式"""
    
    def __init__(self, db_path: str = "data/rag_data.db"):
        self.db_path = db_path
        self._init_processed_table()
    
    def _init_processed_table(self):
        """初始化标准格式表"""
        with sqlite3.connect(self.db_path) as conn:
            conn.executescript("""
                CREATE TABLE IF NOT EXISTS processed_data (
                    id TEXT PRIMARY KEY,
                    question TEXT NOT NULL,
                    question_type TEXT DEFAULT 'factual',
                    domain TEXT DEFAULT '其他',
                    difficulty TEXT DEFAULT 'medium',
                    contexts TEXT,  -- JSON
                    answer TEXT,
                    ground_truth TEXT,
                    metadata TEXT,  -- JSON
                    evaluation TEXT,  -- JSON
                    created_at TEXT DEFAULT CURRENT_TIMESTAMP,
                    processing_stage TEXT DEFAULT 'raw'
                );
                
                CREATE INDEX IF NOT EXISTS idx_proc_stage 
                    ON processed_data(processing_stage);
                CREATE INDEX IF NOT EXISTS idx_proc_domain 
                    ON processed_data(domain);
            """)
    
    def import_from_raw(self, 
                        source_table: str = "conversations",
                        batch_size: int = 100) -> int:
        """从原始表导入到标准格式"""
        
        with sqlite3.connect(self.db_path) as conn:
            conn.row_factory = sqlite3.Row
            
            # 读取未处理的原始数据
            raw_rows = conn.execute(
                f"""SELECT * FROM {source_table} 
                    WHERE id NOT IN (SELECT json_extract(metadata, '$.raw_id') 
                                     FROM processed_data 
                                     WHERE metadata IS NOT NULL)
                    LIMIT ?""",
                (batch_size,)
            ).fetchall()
            
            imported = 0
            for row in raw_rows:
                # 字段映射
                record = self._map_fields(dict(row))
                
                # 写入标准表
                conn.execute("""
                    INSERT INTO processed_data 
                    (id, question, question_type, domain, difficulty,
                     contexts, answer, ground_truth, metadata, 
                     evaluation, processing_stage)
                    VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
                """, (
                    record['id'],
                    record['question'],
                    record['question_type'],
                    record['domain'],
                    record['difficulty'],
                    json.dumps(record['contexts'], ensure_ascii=False),
                    record['answer'],
                    record['ground_truth'],
                    json.dumps(record['metadata'], ensure_ascii=False),
                    json.dumps(record['evaluation'], ensure_ascii=False),
                    'parsed'
                ))
                imported += 1
            
            conn.commit()
        
        return imported
    
    def _map_fields(self, raw: dict) -> dict:
        """字段映射 — 原始字段 → 标准字段"""
        
        # 生成新 ID 或复用
        new_id = str(uuid.uuid4())
        
        # 解析原始 contexts
        contexts = []
        if raw.get('contexts'):
            try:
                ctx_data = json.loads(raw['contexts'])
                if isinstance(ctx_data, list):
                    contexts = [
                        {
                            "content": c if isinstance(c, str) else c.get('content', ''),
                            "source": c.get('source', 'unknown') if isinstance(c, dict) else 'unknown',
                            "page": c.get('page', 1) if isinstance(c, dict) else 1,
                            "relevance_score": c.get('score', 0.9) if isinstance(c, dict) else 0.9
                        }
                        for c in ctx_data
                    ]
            except:
                contexts = [{"content": str(raw['contexts']), "source": "unknown", "page": 1}]
        
        # 构建 metadata
        metadata = {
            "raw_id": raw.get('id'),
            "timestamp": raw.get('timestamp', datetime.now().isoformat()),
            "model_version": raw.get('model_version', 'v1.0.0'),
            "data_source": raw.get('source', 'unknown'),
            "batch_id": raw.get('upload_batch', 'import_' + datetime.now().strftime('%Y%m%d'))
        }
        
        return {
            "id": new_id,
            "question": raw.get('question', ''),
            "question_type": self._infer_question_type(raw.get('question', '')),
            "domain": raw.get('domain', '其他'),
            "difficulty": raw.get('difficulty', 'medium'),
            "contexts": contexts,
            "answer": raw.get('answer', ''),
            "ground_truth": raw.get('ground_truth', ''),
            "metadata": metadata,
            "evaluation": {}
        }
    
    def _infer_question_type(self, question: str) -> str:
        """推断问题类型"""
        q = question.lower()
        if any(w in q for w in ['什么', '是谁', '哪里', 'when', 'what', 'who']):
            return 'factual'
        elif any(w in q for w in ['比较', '区别', 'vs', 'difference', 'compare']):
            return 'comparative'
        elif any(w in q for w in ['如何', '怎么', '步骤', 'how to', 'procedure']):
            return 'procedural'
        else:
            return 'open'
```

```python
# backend/src/testset/parser.py

import re
import json
from typing import Dict, List
from dataclasses import dataclass

@dataclass
class ParseResult:
    """解析结果"""
    is_valid: bool
    question_clean: str
    contexts_clean: List[dict]
    quality_score: float  # 0-1
    errors: List[str]
    warnings: List[str]

class DataParser:
    """数据解析器"""
    
    def __init__(self):
        self.min_question_length = 5
        self.max_question_length = 2000
    
    def parse(self, record: dict) -> ParseResult:
        """解析单条记录"""
        errors = []
        warnings = []
        
        # 1. 清洗问题
        question_clean = self._clean_text(record.get('question', ''))
        if len(question_clean) < self.min_question_length:
            errors.append(f"问题过短: {len(question_clean)} 字符")
        if len(question_clean) > self.max_question_length:
            warnings.append(f"问题过长: {len(question_clean)} 字符")
        
        # 2. 清洗上下文
        contexts_clean = []
        for ctx in record.get('contexts', []):
            clean_ctx = {
                "content": self._clean_text(ctx.get('content', '')),
                "source": ctx.get('source', 'unknown'),
                "page": ctx.get('page', 1),
                "relevance_score": ctx.get('relevance_score', 0.9)
            }
            if len(clean_ctx['content']) > 10:  # 有效内容
                contexts_clean.append(clean_ctx)
            else:
                warnings.append(f"上下文片段过短，已过滤")
        
        # 3. 质量评分
        quality_score = self._calculate_quality(
            question_clean, 
            contexts_clean,
            record.get('answer', ''),
            record.get('ground_truth', '')
        )
        
        is_valid = len(errors) == 0 and quality_score > 0.5
        
        return ParseResult(
            is_valid=is_valid,
            question_clean=question_clean,
            contexts_clean=contexts_clean,
            quality_score=quality_score,
            errors=errors,
            warnings=warnings
        )
    
    def _clean_text(self, text: str) -> str:
        """文本清洗"""
        if not text:
            return ""
        # 去除多余空白
        text = re.sub(r'\s+', ' ', text)
        # 去除特殊字符
        text = re.sub(r'[\x00-\x08\x0b\x0c\x0e-\x1f]', '', text)
        return text.strip()
    
    def _calculate_quality(self, 
                           question: str, 
                           contexts: List[dict],
                           answer: str,
                           ground_truth: str) -> float:
        """计算质量评分"""
        scores = []
        
        # 问题质量 (0-0.3)
        q_score = min(len(question) / 100, 0.3)
        scores.append(q_score)
        
        # 上下文质量 (0-0.3)
        ctx_score = min(len(contexts) * 0.1, 0.3)
        scores.append(ctx_score)
        
        # 答案质量 (0-0.2)
        a_score = min(len(answer) / 500, 0.2) if answer else 0
        scores.append(a_score)
        
        # 标准答案质量 (0-0.2)
        gt_score = 0.2 if ground_truth else 0
        scores.append(gt_score)
        
        return sum(scores)
```

```python
# backend/src/testset/builder.py

import sqlite3
import json
import random
from typing import List, Dict, Optional
from collections import defaultdict

class TestSetBuilder:
    """测试集构建器"""
    
    def __init__(self, db_path: str = "data/rag_data.db"):
        self.db_path = db_path
    
    def build_testset(self,
                      output_prefix: str = "data/testset",
                      golden_size: int = 50,
                      validation_size: int = 200,
                      stress_size: int = 50) -> Dict[str, str]:
        """构建分层测试集"""
        
        # 读取高质量数据
        records = self._load_validated_records(min_quality=0.7)
        
        # 去重
        records = self._deduplicate(records, threshold=0.95)
        
        # 分类
        classified = self._classify_records(records)
        
        # 分层采样
        golden = self._sample_stratified(classified, golden_size, 
                                          priority='quality')
        validation = self._sample_stratified(classified, validation_size,
                                            exclude=golden)
        stress = self._build_stress_set(classified, stress_size)
        
        # 导出
        paths = {}
        paths['golden'] = self._export_jsonl(golden, 
                                              f"{output_prefix}_golden.jsonl")
        paths['validation'] = self._export_jsonl(validation,
                                                  f"{output_prefix}_validation.jsonl")
        paths['stress'] = self._export_jsonl(stress,
                                              f"{output_prefix}_stress.jsonl")
        
        # 更新数据库状态
        self._mark_as_testset([r['id'] for r in golden + validation + stress])
        
        return paths
    
    def _load_validated_records(self, min_quality: float = 0.7) -> List[dict]:
        """加载已验证的高质量数据"""
        with sqlite3.connect(self.db_path) as conn:
            conn.row_factory = sqlite3.Row
            rows = conn.execute(
                """SELECT * FROM processed_data 
                   WHERE processing_stage = 'validated'
                   AND json_extract(metadata, '$.quality_score') >= ?
                   ORDER BY json_extract(metadata, '$.quality_score') DESC""",
                (min_quality,)
            ).fetchall()
            return [dict(row) for row in rows]
    
    def _deduplicate(self, records: List[dict], 
                     threshold: float = 0.95) -> List[dict]:
        """语义去重 — 简单版：基于问题文本相似度"""
        # 教学项目简化：使用字符串相似度
        # 生产环境应使用嵌入相似度
        
        unique = []
        for r in records:
            is_dup = False
            for u in unique:
                sim = self._text_similarity(r['question'], u['question'])
                if sim > threshold:
                    is_dup = True
                    break
            if not is_dup:
                unique.append(r)
        
        return unique
    
    def _text_similarity(self, a: str, b: str) -> float:
        """简单文本相似度（Jaccard）"""
        set_a = set(a.lower().split())
        set_b = set(b.lower().split())
        if not set_a or not set_b:
            return 0.0
        intersection = set_a & set_b
        union = set_a | set_b
        return len(intersection) / len(union)
    
    def _classify_records(self, records: List[dict]) -> Dict[str, List[dict]]:
        """按领域分类"""
        classified = defaultdict(list)
        for r in records:
            domain = r.get('domain', '其他')
            classified[domain].append(r)
        return dict(classified)
    
    def _sample_stratified(self, 
                           classified: Dict[str, List[dict]],
                           total_size: int,
                           exclude: Optional[List[dict]] = None,
                           priority: str = 'balanced') -> List[dict]:
        """分层采样"""
        exclude_ids = {r['id'] for r in (exclude or [])}
        
        # 过滤已排除
        available = {}
        for domain, records in classified.items():
            available[domain] = [r for r in records if r['id'] not in exclude_ids]
        
        # 计算每类采样数
        domains = list(available.keys())
        per_domain = total_size // len(domains)
        
        sampled = []
        for domain in domains:
            pool = available[domain]
            if priority == 'quality':
                # 按质量排序取前N
                pool = sorted(pool, 
                    key=lambda r: json.loads(r.get('metadata', '{}')).get('quality_score', 0),
                    reverse=True)
            
            n = min(per_domain, len(pool))
            sampled.extend(random.sample(pool, n) if priority == 'balanced' 
                          else pool[:n])
        
        # 补足数量
        all_available = [r for records in available.values() for r in records]
        remaining = [r for r in all_available if r['id'] not in {s['id'] for s in sampled}]
        
        if len(sampled) < total_size and remaining:
            need = total_size - len(sampled)
            sampled.extend(random.sample(remaining, min(need, len(remaining))))
        
        return sampled[:total_size]
    
    def _build_stress_set(self, classified: Dict[str, List[dict]], 
                          size: int) -> List[dict]:
        """构建压力测试集 — 边界场景"""
        stress_cases = []
        
        # 收集边界数据
        for domain, records in classified.items():
            for r in records:
                metadata = json.loads(r.get('metadata', '{}'))
                # 选择：短问题、长问题、无上下文、无答案
                if (len(r.get('question', '')) < 10 or
                    len(r.get('question', '')) > 500 or
                    not r.get('contexts') or
                    not r.get('answer')):
                    stress_cases.append(r)
        
        # 随机采样
        if len(stress_cases) > size:
            return random.sample(stress_cases, size)
        return stress_cases
    
    def _export_jsonl(self, records: List[dict], path: str) -> str:
        """导出 JSONL"""
        with open(path, 'w', encoding='utf-8') as f:
            for r in records:
                # 解析 JSON 字段
                export_record = {
                    "id": r['id'],
                    "question": r['question'],
                    "question_type": r['question_type'],
                    "domain": r['domain'],
                    "difficulty": r['difficulty'],
                    "contexts": json.loads(r.get('contexts', '[]')),
                    "answer": r.get('answer', ''),
                    "ground_truth": r.get('ground_truth', ''),
                    "metadata": json.loads(r.get('metadata', '{}'))
                }
                f.write(json.dumps(export_record, ensure_ascii=False) + '\n')
        
        return path
    
    def _mark_as_testset(self, ids: List[str]):
        """标记为测试集数据"""
        with sqlite3.connect(self.db_path) as conn:
            for id_ in ids:
                conn.execute(
                    "UPDATE processed_data SET processing_stage = 'testset' WHERE id = ?",
                    (id_,)
                )
            conn.commit()
```

### 4.1 RAGAS 评估

#### 评估指标

| 指标名 | 含义 | 计算方式 | 目标值 | 优先级 |
|--------|------|---------|--------|--------|
| **Faithfulness** | 回答是否忠实于检索到的上下文 | 回答中的事实可被上下文支撑的比例 | > 0.8 | P0 |
| **Answer Relevance** | 回答与问题的相关度 | 回答与问题的语义相似度 | > 0.8 | P0 |
| **Context Precision** | 检索结果中相关片段的比例 | 相关chunk数 / 总chunk数 | > 0.7 | P0 |
| **Context Recall** | 应被检索到的相关内容实际被检索到的比例 | 检索到的相关内容 / 所有相关内容 | > 0.6 | P1 |
| **RAGAS Score** | 综合评分（加权平均） | 0.4*Faithfulness + 0.3*Relevance + 0.3*Precision | > 0.75 | P0 |

```python
# backend/src/evaluation/ragas_eval.py

import json
import sqlite3
from typing import List, Dict, Optional
from datasets import Dataset
from ragas import evaluate
from ragas.metrics import (
    faithfulness,
    answer_relevancy,
    context_precision,
    context_recall,
)
from langchain_openai import ChatOpenAI

class RAGASEvaluator:
    """RAGAS 评估器 — 教学项目轻量版"""
    
    def __init__(self, 
                 api_key: str,
                 model: str = "openai/gpt-4o-mini",
                 db_path: str = "data/rag_data.db"):
        self.db_path = db_path
        
        # LLM 配置（OpenRouter）
        self.llm = ChatOpenAI(
            model=model,
            openai_api_base="https://openrouter.ai/api/v1",
            openai_api_key=api_key,
        )
        
        # 评估指标
        self.metrics = [
            faithfulness,
            answer_relevancy,
            context_precision,
            context_recall,
        ]
        
        self._init_results_table()
    
    def _init_results_table(self):
        """初始化评估结果表"""
        with sqlite3.connect(self.db_path) as conn:
            conn.executescript("""
                CREATE TABLE IF NOT EXISTS evaluation_results (
                    id TEXT PRIMARY KEY,
                    testset_version TEXT,
                    testset_type TEXT,  -- golden|validation|stress
                    record_id TEXT,
                    faithfulness REAL,
                    answer_relevancy REAL,
                    context_precision REAL,
                    context_recall REAL,
                    ragas_score REAL,
                    passed BOOLEAN,
                    evaluated_at TEXT DEFAULT CURRENT_TIMESTAMP,
                    metadata TEXT
                );
                
                CREATE INDEX IF NOT EXISTS idx_eval_version 
                    ON evaluation_results(testset_version);
                CREATE INDEX IF NOT EXISTS idx_eval_time 
                    ON evaluation_results(evaluated_at);
            """)
    
    def evaluate_testset(self, 
                         testset_path: str,
                         testset_version: str,
                         testset_type: str = "validation") -> Dict:
        """评估测试集"""
        
        # 1. 加载测试集
        records = self._load_testset(testset_path)
        
        # 2. 构建 Dataset
        dataset = self._build_dataset(records)
        
        # 3. 运行 RAGAS
        result = evaluate(
            dataset=dataset,
            metrics=self.metrics,
            llm=self.llm,
        )
        
        # 4. 处理结果
        df = result.to_pandas()
        
        # 5. 计算综合得分
        df['ragas_score'] = (
            0.4 * df['faithfulness'] + 
            0.3 * df['answer_relevancy'] + 
            0.3 * df['context_precision']
        )
        
        # 6. 存储结果
        self._save_results(df, testset_version, testset_type, records)
        
        # 7. 汇总统计
        summary = {
            "testset_version": testset_version,
            "testset_type": testset_type,
            "total": len(df),
            "avg_faithfulness": float(df['faithfulness'].mean()),
            "avg_relevancy": float(df['answer_relevancy'].mean()),
            "avg_precision": float(df['context_precision'].mean()),
            "avg_recall": float(df['context_recall'].mean()),
            "avg_ragas_score": float(df['ragas_score'].mean()),
            "passed": int((df['ragas_score'] > 0.75).sum()),
            "failed": int((df['ragas_score'] <= 0.75).sum()),
            "pass_rate": float((df['ragas_score'] > 0.75).mean()),
        }
        
        return summary
    
    def _load_testset(self, path: str) -> List[dict]:
        """加载测试集"""
        records = []
        with open(path, 'r', encoding='utf-8') as f:
            for line in f:
                records.append(json.loads(line.strip()))
        return records
    
    def _build_dataset(self, records: List[dict]) -> Dataset:
        """构建 RAGAS Dataset"""
        data = {
            "question": [],
            "answer": [],
            "contexts": [],
            "ground_truth": [],
        }
        
        for r in records:
            data["question"].append(r["question"])
            data["answer"].append(r.get("answer", ""))
            
            # contexts 需要是 List[str]
            contexts = r.get("contexts", [])
            if isinstance(contexts, list) and contexts:
                if isinstance(contexts[0], dict):
                    context_texts = [c.get("content", "") for c in contexts]
                else:
                    context_texts = contexts
            else:
                context_texts = []
            data["contexts"].append(context_texts)
            
            data["ground_truth"].append(r.get("ground_truth", ""))
        
        return Dataset.from_dict(data)
    
    def _save_results(self, 
                      df, 
                      testset_version: str,
                      testset_type: str,
                      records: List[dict]):
        """存储评估结果"""
        import uuid
        
        with sqlite3.connect(self.db_path) as conn:
            for idx, row in df.iterrows():
                conn.execute("""
                    INSERT INTO evaluation_results 
                    (id, testset_version, testset_type, record_id,
                     faithfulness, answer_relevancy, context_precision,
                     context_recall, ragas_score, passed, metadata)
                    VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
                """, (
                    str(uuid.uuid4()),
                    testset_version,
                    testset_type,
                    records[idx]["id"],
                    float(row['faithfulness']),
                    float(row['answer_relevancy']),
                    float(row['context_precision']),
                    float(row['context_recall']),
                    float(row['ragas_score']),
                    bool(row['ragas_score'] > 0.75),
                    json.dumps({"question": records[idx]["question"][:100]})
                ))
            conn.commit()
    
    def get_recent_evaluations(self, 
                               testset_version: Optional[str] = None,
                               limit: int = 10) -> List[dict]:
        """获取最近评估结果"""
        with sqlite3.connect(self.db_path) as conn:
            conn.row_factory = sqlite3.Row
            
            if testset_version:
                rows = conn.execute(
                    """SELECT * FROM evaluation_results 
                       WHERE testset_version = ?
                       ORDER BY evaluated_at DESC LIMIT ?""",
                    (testset_version, limit)
                ).fetchall()
            else:
                rows = conn.execute(
                    """SELECT * FROM evaluation_results 
                       ORDER BY evaluated_at DESC LIMIT ?""",
                    (limit,)
                ).fetchall()
            
            return [dict(row) for row in rows]
    
    def evaluate_single(self, 
                        question: str, 
                        answer: str, 
                        contexts: List[str],
                        ground_truth: str = "") -> Dict:
        """单条评估 — 用于实时反馈"""
        dataset = Dataset.from_dict({
            "question": [question],
            "answer": [answer],
            "contexts": [contexts],
            "ground_truth": [ground_truth],
        })
        
        result = evaluate(
            dataset=dataset,
            metrics=self.metrics,
            llm=self.llm,
        )
        
        df = result.to_pandas()
        
        return {
            "faithfulness": float(df['faithfulness'].iloc[0]),
            "answer_relevancy": float(df['answer_relevancy'].iloc[0]),
            "context_precision": float(df['context_precision'].iloc[0]),
            "context_recall": float(df['context_recall'].iloc[0]),
            "ragas_score": float(
                0.4 * df['faithfulness'].iloc[0] +
                0.3 * df['answer_relevancy'].iloc[0] +
                0.3 * df['context_precision'].iloc[0]
            ),
        }
```

```python
                    AVG(context_precision) as avg_precision,
                    AVG(context_recall) as avg_recall,
                    AVG(ragas_score) as avg_ragas_score,
```

```python
# backend/src/feedback/observer.py

import sqlite3
import json
import logging
from datetime import datetime
from typing import Dict, List, Optional

# 配置日志
logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    handlers=[
        logging.FileHandler('logs/evaluation.log'),
        logging.StreamHandler()
    ]
)
logger = logging.getLogger('ragas-observer')

class SimpleObserver:
    """轻量可观测 — 日志 + SQLite"""
    
    def __init__(self, db_path: str = "data/rag_data.db"):
        self.db_path = db_path
        self._init_observability_table()
    
    def _init_observability_table(self):
        """初始化可观测表"""
        with sqlite3.connect(self.db_path) as conn:
            conn.executescript("""
                CREATE TABLE IF NOT EXISTS observability_logs (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    timestamp TEXT DEFAULT CURRENT_TIMESTAMP,
                    level TEXT,  -- INFO|WARN|ERROR
                    component TEXT,  -- evaluation|collection|testset
                    operation TEXT,
                    message TEXT,
                    metadata TEXT  -- JSON
                );
                
                CREATE INDEX IF NOT EXISTS idx_obs_component 
                    ON observability_logs(component);
                CREATE INDEX IF NOT EXISTS idx_obs_level 
                    ON observability_logs(level);
            """)
    
    def log(self, 
            level: str,
            component: str,
            operation: str,
            message: str,
            metadata: Optional[Dict] = None):
        """记录可观测日志"""
        
        # 打印日志
        log_func = getattr(logger, level.lower(), logger.info)
        log_func(f"[{component}] {operation}: {message}")
        
        # 存储到 SQLite
        with sqlite3.connect(self.db_path) as conn:
            conn.execute("""
                INSERT INTO observability_logs 
                (level, component, operation, message, metadata)
                VALUES (?, ?, ?, ?, ?)
            """, (
                level,
                component,
                operation,
                message,
                json.dumps(metadata or {}, ensure_ascii=False)
            ))
            conn.commit()
    
    def get_logs(self,
                 component: Optional[str] = None,
                 level: Optional[str] = None,
                 limit: int = 100) -> List[dict]:
        """查询日志"""
        with sqlite3.connect(self.db_path) as conn:
            conn.row_factory = sqlite3.Row
            
            query = "SELECT * FROM observability_logs WHERE 1=1"
            params = []
            
            if component:
                query += " AND component = ?"
                params.append(component)
            if level:
                query += " AND level = ?"
                params.append(level)
            
            query += " ORDER BY timestamp DESC LIMIT ?"
            params.append(limit)
            
            rows = conn.execute(query, params).fetchall()
            return [dict(row) for row in rows]
    
    def get_error_rate(self, 
                       component: str,
                       hours: int = 24) -> float:
        """获取错误率"""
        since = (datetime.now() - timedelta(hours=hours)).isoformat()
        
        with sqlite3.connect(self.db_path) as conn:
            total = conn.execute(
                "SELECT COUNT(*) FROM observability_logs WHERE component = ? AND timestamp > ?",
                (component, since)
            ).fetchone()[0]
            
            errors = conn.execute(
                "SELECT COUNT(*) FROM observability_logs WHERE component = ? AND level = 'ERROR' AND timestamp > ?",
                (component, since)
            ).fetchone()[0]
            
            return errors / total if total > 0 else 0.0
```

| **提示词长度** | Token数 | 计数 | 过长 → 截断/压缩上下文 |
| **提示词结构** | 模板匹配度 | 正则检查 | 不匹配 → 告警 + 回退默认模板 |
| **提示词效果** | 同一提示词不同输入的得分方差 | RAGAS评估 | 方差大 → 提示词需优化 |
| **版本对比** | A/B测试得分 | 分流评估 | 新版本得分低 → 回滚 |

**提示词版本管理：**

```python
# backend/src/prompts/registry.py

PROMPT_VERSIONS = {
    "v1.0.0": {
        "template": "基于以下文档回答问题：\n{context}\n\n问题：{question}",

    },

        "template": "你是专业知识助手。基于提供的文档，给出清晰准确的回答。\n\n文档：\n{context}\n\n问题：{question}",
        "ragas_score": 0.78,
        "usage_count": 3000,
        "status": "canary",  # canary | stable | deprecated
    },
}

# 自动选择：canary 流量 10%，stable 流量 90%
# 若 canary 的 RAGAS score 连续7天 > stable，则提升为 stable
```
